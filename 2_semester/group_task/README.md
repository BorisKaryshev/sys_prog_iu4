# Общая инфа

Есть самолет, управляющая система на нем - Linux

Нам нужно написать написать модуль ядра, которым мы сможем влиять на датчик (другой модуль)
Наш модуль ядра должен быть скрытным.

Наш процесс (порожденный нашим модулем ядра) не должен быть виден утилитами:
- `ps aux`
- `df -h`
- `netstat`
- `tulplan`
- `top`
- `trace` (`strace` etc)


# Конкретно наше условие
Нужно сделать модуль ядра
Его нужно скрыть: он не должен светиться в `lsmod` или `cat /proc/modules`
Модуль должен создавать процесс, который будет невиден через `ps auxfww`
